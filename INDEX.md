# INDEX

## Top-level Areas

- `schemas/` : optional schema definitions for structured objects
- `templates/` : reusable templates for objects and mappings
- `organization/` : org-level examples and policies
- `teams/` : team-level examples
- `people/` : people objects
- `agents/` : agent objects
- `projects/` : project examples
- `resources/` : devices, tools, repositories, datasets, services
- `matrix/` : cross-object relationship maps
- `docs/` : recovery, inheritance, and writing rules
